【小売業者様向け】POS連携SDK 組み込みガイド

本SDK（pos_connect_sdk.js）は、
既存のPOSシステムに
「どこでもレジ」を
瞬時に連携・導入する
スクリプトです。
貴社のPOSプログラミングに
このSDKを1行コピーして
組み込むだけで、
既存レジの削減、
レジ待ち行列の解消、
そして商品マスターの
自動クラウド同期が
即座に実現し、
お客様は自分のスマホで
並ばずに決済
（モバイルチェックアウト）
ができるようになります。

既存のPOSレジへの組み込み例

貴社の既存POSレジのプログラム内で
本SDKを読み込ませるだけで、
店舗の商品データとクラウドが自動同期され
顧客用スマホレジ（どこでもレジ）が稼働開始します。

<!-- 1. どこでもレジ連携SDKを読み込む --> <script src="https://貴社のAppRunnerドメイン/pos_connect_sdk.js"></script> <!-- 2. POSの起動時・更新時に商品マスターを同期する --> <script> // 専用APIキーを使ってどこでもレジ・システムと接続 const anywhereRegi = new AnywhereRegiConnect('発行された専用APIキー'); // 既存POSの商品マスター // （価格・在庫）をクラウドと // 全自動同期します anywhereRegi.syncProducts(); // これで準備は完了です！ // あとは「店舗の入り口」や // 「カート」にあるQRから、 // お客様が自身のスマホで // 自由に決済（どこでもレジ） // できるようになり、 // レジ待ち行列を回避できます。 </script>

導入による3つのメリット

  1. レジハードウェアの削減: お客様自身のスマートフォンがレジの代わりとなるため、高額なPOSレジ端末の増設が一切不要になります。
  2. レジ待ち行列の解消: 混雑時でも、お客様が自分でバーコードをスキャンして決済（モバイルチェックアウト）できるため、店舗の回転率と顧客満足度が劇的に向上します。
  3. シームレスな体験: お客様は専用アプリをインストールすることなく、店舗の入り口やカートにあるQRコードを読み取るだけで、ブラウザから直接決済（Apple Pay / Google Pay等）が可能です。
