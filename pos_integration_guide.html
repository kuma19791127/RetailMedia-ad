【小売業者様向け】POS連携SDK 組み込みガイド

本SDK（pos_connect_sdk.js）は、
既存のPOSシステムに
「どこでもレジ」を
瞬時に連携・導入する
スクリプトです。
貴社のPOSプログラミングに
このSDKを1行コピーして
組み込むだけで、
既存レジの削減、
レジ待ち行列の解消、
そして商品マスターの
自動クラウド同期が
即座に実現し、
お客様は自分のスマホで
並ばずに決済
（モバイルチェックアウト）
ができるようになります。

既存のPOSレジへの組み込み例

貴社の既存POSレジのプログラム内で
本SDKを読み込ませるだけで、
店舗の商品データと
クラウドが自動同期され
顧客用スマホレジ
（どこでもレジ）が稼働開始します。

<!-- 1. どこでもレジ連携SDKを読み込む --> <script src="https://貴社のAppRunnerドメイン/pos_connect_sdk.js"></script> <!-- 2. POSの起動時・更新時に商品マスターを同期する --> <script> // 専用APIキーを使ってどこでもレジ・システムと接続 const anywhereRegi = new AnywhereRegiConnect('発行された専用APIキー'); // 既存POSの商品マスター // （価格・在庫）をクラウドと // 全自動同期します anywhereRegi.syncProducts(); // これで準備は完了です！ // あとは「店舗の入り口」や // 「カート」にあるQRから、 // お客様が自身のスマホで // 自由に決済（どこでもレジ） // できるようになり、 // レジ待ち行列を回避できます。 </script>

導入による3つのメリット

1. レジハードウェアの削減 お客様自身のスマホが
レジの代わりとなるため、
高額なPOSレジ端末の
増設が一切不要になります。
2. レジ待ち行列の解消 混雑時でもお客様が自分で
バーコードをスキャンして
決済できるため、
店舗の回転率と顧客満足度が
劇的に向上します。
3. シームレスな体験 専用アプリのインストールは
一切不要です。
入り口のQRを読み取るだけで
ブラウザから直接決済
（Apple / Google Pay）
が可能です。

👑 絞り込まれた上位顧客（VIP）の
特権的な買い物ステップ

1. 【特別感の演出】
VIP限定の案内状を受け取る 店舗から厳選された
上位顧客のご自宅へ特別な
ダイレクトメール（レター）や
会員限定の案内が届きます。
そこには「VIP様限定：
レジに並ばずお買い物が
できる『どこでもレジ』のご案内」
として
専用のQRコードが
印字されています。
2. 【来店・起動】
シークレットレジを立ち上げ お客様が来店された際
ご自身のスマホでその
「レターに記載された
専用QRコード」を読み取ります。
アプリは不要で
瞬時にお客様専用の
「モバイル決済画面」が
ブラウザ上に立ち上がります。
（まさにVIPパスのよう役割です）
3. 【買物中】
商品を自分のスマホでスキャン 一般のお客様がカゴを持って
レジ待ちをしている中
VIP顧客は棚から取った商品を
自身のスマホカメラで
「ピッ」と読み込みます。
音声読み上げと共に
手元の画面で合計金額を
リアルタイムに確認しながら
スマートに買い物が進みます。
4. 【お会計】
行列を横目に1タップ決済 買い物が終わったら
レジには向かわず
店内のどこにいても
画面のボタンを押し
Apple Pay等の顔認証で
一瞬で決済完了。
