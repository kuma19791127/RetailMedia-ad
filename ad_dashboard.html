📡 External API Endpoints

POST /api/external/v1/event

POS Connect: Send sales/conversion data to update dashboard metrics (Real-time CPA).
Payload: { "transactionId": "TX123", "amount": 1500, "items": [...] }

GET /api/external/v1/inventory

DSP Connect: Retrieve available ad slots and estimated CPM/Impressions.

📦 POS Connect SDK (JavaScript)

Easily integrate retail-ad conversion tracking into your web-based POS or E-Commerce site.

// 1. Install via NPM (Coming Soon)
$ npm install @retail-ad/pos-connect-sdk
// 2. Or Include Script (Direct Download)
<script src="/pos_connect_sdk.js"></script>
// 3. Usage
const client = new retail-adPOS('LITEAD_API_KEY_...');
client.trackSale({
  transactionId: 'ORDER_001',
  amount: 2500,
  items: ['premium_plan']
}).then(res => console.log('Tracked!', res));

💡 よくある質問

Q. YouTubeが再生されません。
A. ブラウザのセキュリティ設定により、初回のみ画面のどこかをクリックする必要があります。

Q. 請求のタイミングは？
A. 広告が表示された回数に応じて、月末にまとめて請求書が発行されます。

Q. 配信エリアを変更したい。
A. ダッシュボードの「配信時間帯」設定から変更可能ですが、エリア設定は管理者への申請が必要です。
