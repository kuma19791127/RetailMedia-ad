サイネージセットアップ手順

■ Android端末（既存Androidパネル等）の場合 1. サイネージ用Android端末のブラウザ等から、以下のURLを開いてプレイヤーアプリ (APK) をダウンロードし、インストールします。
📥 アプリ(APK)をダウンロード
2. インストール完了後、以下の【自動セットアップ起動リンク】をAndroid端末のブラウザ等でタップして起動します。
litead://setup?storeId=1000001
3. アプリが自動で起動し、店舗IDが自動入力された状態で、即座にサイネージのフルスクリーン再生が開始されます。

■ Windows PCの場合 1. 端末詳細の「簡易スタートガイド」または「設定手順書」のバッチファイルをダウンロードします。
2. 保存したバッチファイルを右クリックし、「管理者として実行」してください。

■ セキュリティ設定を行わない場合 1. 簡易スタートガイドを開くか、以下のURLをブラウザ（Google Chrome推奨）で開いてください。

Google Ad Manager (MCM) / AdSense 連携設定

「親アカウント（システム運営者）」
のネットワークに参加することで、
面倒なAdSense審査や設定をスキップして、
すぐに各サイネージパネル（端末）
ごとの広告収益化を開始できます。
連携が完了すると、自動的に各端末固有の
「広告ユニットID」が生成され、
キーバリュー（Key-Values: ?terminal_id=
[固有ID]）
を通じて配信と収益計測がスタートします。

POS・データ連携設定

店舗の売上データをシステムと連携し、広告主には自社商品や種類の詳細リアルタイムデータを提供　ストアには成果報酬型広告を稼働させます。

🟢 POS接続SDK (pos_connect_sdk.js) によるレジ連携

東芝テックや富士通などの独自POSシステムの場合、システムに同梱されている軽量な pos_connect_sdk.js をレジのプログラムに組み込むことで連携が完了します。

// 【貼り付け位置】既存POSレジの「会計処理完了・確定イベント」のハンドラ内に以下を組み込みます
function onPaymentSuccess(transactionData) {
    // リテアドへの売上自動同期を実行
    liteAd.trackSale({
        transactionId: transactionData.id,  // 決済伝票番号
        amount: transactionData.totalPrice,  // 会計総額
        items: transactionData.janCodes      // 購入商品のJANコード配列 (例: ['4901335109634'])
    });
}

🔄 POS売上データの定期自動送信機能 (バックグラウンド同期)

店舗システム（store_portal）およびPOS（anywhere_regi）は「販売データを記録・送信する」役割に特化し、5分ごとに自動送信を行います。広告マッチング等の重い処理は本部サーバー（Admin）で行うことで、店舗端末の負荷を完全に下げます。
