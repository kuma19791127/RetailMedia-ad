💻 1. Windows PCでの
自動起動・全画面設定

🔒 ご導入時のセキュリティに関する
ご案内（OS別の選択） 本サービスは Windows PC / Android の
両方で運用可能です。店舗の
運用ポリシーに合わせてお選びください。
・Windows PC方式：既存のパソコンや
小型PCで手軽にスタートアップ起動が
可能です。※キーボードやUSB機器が
露出している環境では、いたずら防止対策として
「Androidアプリ方式（方式2）」の
併用をご検討ください。
・Androidアプリ方式
（方式2・より高セキュリティ）：
OS層での「完全な物理セキュリティブロック
（USB無効化、キーボード誤操作の
完全ロック、MDM遠隔制御）」が
機能するため、不特定多数の来店客が
触れる環境や、法人・大店舗での
セキュリティ重視の運用に最も適しています。

Windows PCを使用して
自動起動設定を行う場合は、
以下の手順を実行してください。

⚙️ 方法A：自動起動設定ツールの
実行（簡易設定）

PC起動時に自動でChromeを
キオスクモード（全画面）で
起動するツールを用意しています。

⏬ Windows自動起動ツールDL
💡 設定ファイルの適用方法 ダウンロードしたバッチファイルを
右クリックし、「管理者として実行」して
ください。自動的にWindowsの
スタートアップへ登録されます。
解除する場合は、再度実行して
画面表示に従って解除できます。

🛠️ 方法B：手動での全画面・
自動起動設定

  1. Chromeのショートカットアイコンを右クリックし、「プロパティ」を開きます。
  2. 「リンク先」の記述の末尾に、半角スペースを空けて以下のように追記します。 --kiosk https://nsg3hyme2k.us-east-1.awsapprunner.com/signage_player.html
  3. 修正したショートカットアイコンをコピーします。
  4. キーボードの「Windowsキー + Rキー」を同時に押し、入力欄に shell:startup と入力してEnterを押します。
  5. 開いたフォルダ（スタートアップフォルダ）に、コピーしたショートカットを貼り付けます。これでPC起動時に自動実行されます。

🤖 2. Android端末・
スマートモニターでの
自動起動設定

Android端末では、電源オン時に
自動でブラウザ（Chrome）を開いて
Webプレイヤーを全画面表示するために、
以下のステップで設定を行ってください。

📲 ステップ1. 自動起動アプリの
インストール

AndroidにはWindowsのような
バッチファイルがないため、自動起動を
支援するアプリを利用します。
Google Playストアより無料アプリを
インストールしてください。

Google Playで「AutoStart」を表示

⚙️ ステップ2. アプリ自動起動の設定
（AutoStartの設定）

  1. インストールした「AutoStart」アプリを起動します。
  2. 「AutoStart Service」を ON（有効）にします。
  3. 「Add Applications」または「アプリ追加」ボタンをタップし、アプリ一覧から 「Chrome」 を選択して登録します。
  4. 端末を再起動し、電源オン時に自動でChromeが起動することを確認してください。

📌 ステップ3. プレイヤー表示と
画面ピン留め（固定化）

  1. Chromeを起動し、サイネージプレイヤーURLを開きます。 https://nsg3hyme2k.us-east-1.awsapprunner.com/signage_player.html
  2. プレイヤー画面が開いたら、Androidのタスク一覧（起動中のアプリ一覧画面）を開きます。
  3. Chromeアプリのアイコンをタップ、または長押しし、表示されるメニューから 「ピン留め（または画面固定）」 を選択します。
  4. これで、誤タップや操作によってプレイヤー画面が閉じたり、他の画面に切り替わったりするのを防ぎ、専用サイネージとして安全に稼働させることができます。
